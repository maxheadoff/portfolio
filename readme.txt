Программа "Банкомат"

Банкомат позволяет принимать и выдавать деньги, отображать состояние счета.
Купюры могут быть разного достоинства (10,50,100,500,1000,5000 ). Банкомат имеет
ограничение по количеству хранимых в нем купюр, которое указывается в файле настроек CashMachine.vshost.exe.config парематр "cashCapacity".
Перед выдачей купюр банкомат предлагает выбор купюр предпочитаемого достоинства из доступных.

Алгоритм подбора купюр имеет особенность: 
т.к. предпочитаемая купюра имеет преимущество при выборе, не рассматривается вариант выдачи требуемой суммы более крупной купюрой,
пока в банкомате есть предпочитаемая купюра.

Проект собран в VS2012 .net framework 4.5
